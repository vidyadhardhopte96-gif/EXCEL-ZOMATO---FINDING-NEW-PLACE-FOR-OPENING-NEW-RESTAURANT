# EXCEL-ZOMATO---FINDING-NEW-PLACE-FOR-OPENING-NEW-RESTAURANT
Out of the raw data consisting of 9544 Rows, 27 Columns, 257634 data cells. Analysis was performed of various countires having details of address, to see which place is good enough to open a new restaurant. Various anaylsis such as data clearing, data transformation, modelling, data visualization, pivot tables, aggregate functions, etc were used to perfom data analysis.
